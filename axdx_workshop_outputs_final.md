# Problem Definition

## 1. 주제
소비패턴 분석 금융 코치 앱

## 2. 문제 상황
[월급날] 예산 설계 없이 소비를 시작함
[중반] 배달·쇼핑 중심의 충동 소비가 반복되어 무분별한 변동 지출이 발생함
[월말] 카드 명세서를 확인하지만 소비 원인 분석 없이 끝남.
소비 데이터는 쌓이지만 실제 행동 개선으로 이어지지 않음

## 3. 핵심 사용자
금융 관리 경험이 적은 사회초년생, 재테크 의지는 높으나 불확실한 재정 목표를 설정해 실행력이 낮은 직장인

## 4. 사용자가 겪는 불편
어디에 돈을 많이 쓰는지 한눈에 파악하기 어려움
단순 지출 내역만 제공되어 소비 습관 개선 방향을 알기 어려움
금융 앱의 복잡한 데이터와 그래프로 인해 피로감을 느낌
소비 통제가 안 된다는 불안감과 재정 스트레스를 경험함

## 5. 현재 대안과 한계
카드사 앱이나 가계부 앱에서 소비 내역만 확인함
엑셀·메모장으로 직접 예산을 관리하지만 지속성이 낮음
기존 금융 앱은 데이터 나열 중심이라 개인 맞춤형 피드백이 부족함
소비 분석 결과가 실제 행동 변화까지 연결되지 못함

## 6. 서비스 아이디어
사용자가 영수증을 촬영하거나 소비 내역을 직접 입력하면,
앱이 소비 데이터를 저장하고, AI가 소비 패턴과 과소비 항목을 분석
과소비 카테고리와 반복 소비 패턴을 시각적으로 제공
사용자 소비 습관에 맞춘 저축·절약 솔루션 추천
월 예산 대비 현재 소비 위험도를 실시간 피드백으로 제공
“이번 주 배달비가 평균보다 25% 높아요” 같은 행동 중심 코칭 제공
사용자의 소비 카테고리 비중을 분석하여 식비형·쇼핑형· 교통형·카페형 등 소비패턴에 적합한 카드 혜택 유형 추천 (MVP 단계에서는 실제 카드사 API 연동 없이 더미 카드 데이터를 활용하여 추천 결과를 제공)
최종 MVP에서는 월별 소비 필터링, 선택 월 기준 CSV 다운로드, OCR 결과 수정 후 저장, 전체 데이터 초기화 기능을 추가하여 사용자가 소비 데이터를 더 쉽게 관리할 수 있도록 구현

## 7. 4주 안에 구현 가능한 최소 기능
영수증 촬영 및 OCR 소비 데이터 자동 입력 기능
소비 내역 직접 입력 및 수정 기능
소비 카테고리별 지출 분석 기능
월 예산 대비 소비 현황 시각화, 월별 소비 패턴 시각화 대시보드
AI 기반 간단 소비 피드백 제공(과소비 항목 분석 및 AI 코멘트 제공)
소비 위험 알림 및 절약 추천 문구 출력

## 8. 데이터로 검증할 수 있는 지표
주간 소비 기록 유지율 = 7일 이상 연속 소비 기록 사용자 수 / 전체 사용자 수 × 100
예산 설정 사용자 비율 
예산 달성률 = 예산 이하로 소비한 사용자 수 / 전체 사용자 수 × 100
과소비 카테고리 감소율 = (서비스 사용 전 지출액 - 사용 후 지출액) / 사용 전 지출액 × 100
AI 피드백 조회율
소비 분석 화면 재방문율

## 9. 구현 범위에서 제외할 것
실제 은행·카드사 계좌 연동
자동 결제 및 송금 기능
투자 상품 추천 및 자산 운용 기능
복잡한 금융 상품 비교 기능
프리미엄 유료 결제 시스템
고도화된 AI 예측 모델 및 신용평가 기능

# Persona

- 이름: 박지훈
- 나이: 27
- 전공: 산업경영공학
- 학년: 4학년 (인턴십 중)

## 하루 일과
오전에 출근 준비를 하며 카페 앱으로 커피를 주문한다.  
출근 후에는 업무 중 배달앱이나 간편결제를 자주 사용하고, 점심값과 카페 소비가 반복적으로 발생한다.  
퇴근 후에는 스트레스 해소를 위해 온라인 쇼핑이나 게임, OTT 구독 서비스를 자주 이용한다.  
월급일 직후에는 소비가 증가하지만, 월말에는 생활비 부족을 체감하며 소비를 줄이려 한다.

## 자주 사용하는 도구
카카오뱅크, 토스, 카드사 앱, 배달의 민족, 쿠팡, 카카오톡, 네이버페이

## 목표
본인의 소비 패턴을 파악하고 불필요한 지출을 줄여
월급의 일정 금액을 꾸준히 저축하는 습관을 만들고자 한다.
인턴 종료 후 취업 준비와 미래 생활비를 위한 안정적인 자금 관리를 목표로 한다.

## Pain Point
카드 사용 내역을 통해 어디에 돈을 많이 쓰는지 체감은 되지만, 어느 부분에서 과소비를 하는지 한눈에 파악하기 어려워 정확한 분석이 어렵다.
카드 명세서를 봐도 소비 개선 방법을 모르겠다.
월급이 들어오면 계획 없이 소비하다가 월말에 잔액 부족을 느껴 이전의 무분별한 소비를 후회한다.
기존 가계부 앱은 모든 소비 내역을 직접 입력해야 해서 번거롭고 오래 사용하지 못한다.
또한 영수증이 없거나 OCR 인식이 어려운 소비는 직접 수정할 수 있는 기능도 필요하다고 느낀다.
금융 앱은 기능이 많고 복잡해 필요한 정보만 빠르게 보기 어렵다.

## 서비스 사용 전 행동
카드 사용 내역과 계좌 잔액만 간단히 확인한다.
소비내역에서 각 소비 부분의 정확한 지출 규모는 인지하지 못한다.
할인·쿠폰 알림이 오면 필요하지 않은 소비도 쉽게 발생한다.
월말에 잔액 부족을 느끼고 소비를 반성한다.
가계부 앱을 설치했다가 며칠 뒤 사용을 중단한다.
소비 데이터를 봐도 행동 변화로 이어지지 않는다.

## 서비스 사용 후 기대 변화
영수증 촬영 또는 직접 입력으로 소비 내역을 간편하게 기록할 수 있다.
AI가 기록된 소비 데이터를 분석해 과소비 항목과 소비 패턴을 자동으로 파악하고, “이번 주 배달 소비가 평균보다 높아요”와 같은 직관적인 피드백을 제공한다.
남은 예산, 저축 가능 금액과 소비 위험도를 쉽게 확인할 수 있다.
맞춤형 절약 팁과 소비 피드백으로 행동 개선이 가능해진다.
소비 흐름을 한눈에 확인하며 계획적인 금융 습관을 형성한다.
월별 저축 목표를 설정하고 달성 여부를 쉽게 관리할 수 있다.

## 절대 원하지 않는 것
복잡한 금융 용어와 어려운 그래프
긴 회원가입 및 복잡한 초기 설정
광고성 금융상품 추천
모든 소비 내역을 반드시 직접 입력해야 하는 방식, OCR 인식 실패 시 수정이 어렵거나 다시 처음부터 입력해야 하는 방식
지나치게 잦은 푸시 알림

# PRD

## 문제정의
[월급날] 예산 설계 없이 소비를 시작함
[중반] 배달·쇼핑 중심의 충동 소비가 반복됨
[월말] 카드 명세서를 확인하지만 소비 원인 분석 없이 끝남
소비 데이터는 쌓이지만 실제 행동 개선으로 이어지지 않음

## 목표 사용자
금융 관리 경험이 적은 사회초년생, 재테크 의지는 높으나 불확실한 재정 목표를 설정해 실행력이 낮은 직장인

## 핵심 가치 제안
사용자의 소비 데이터를 AI가 분석하여 과소비 패턴과 저축 가능 영역을 쉽고 직관적으로 보여주고, 행동 가능한 맞춤형 금융 습관 개선 솔루션을 제공한다.

## 핵심 시나리오
사용자가 영수증을 촬영하거나 소비 내역을 직접 입력하면,
앱이 소비 금액·날짜·가맹점·카테고리를 저장한다.
저장된 소비 데이터를 바탕으로 AI가 주요 지출 카테고리, 과소비 항목, 예산 위험도, 맞춤 절약 솔루션을 제공한다. 또한 소비 비중이 가장 높은 카테고리를 기반으로 적합한 카드 혜택 유형을 추천한다.

## MVP 기능 범위
1. 닉네임 입력
2. 월 수입, 월 예산, 저축 목표 입력
3. 소비 내역 직접 입력
4. 소비 내역 수정 및 삭제
5. 영수증 이미지 업로드
6. OCR 기반 소비 금액, 날짜, 카테고리 자동 인식
7. OCR 결과 수정 후 저장
8. LocalStorage 기반 소비 데이터 저장
9. 월별 소비 내역 필터링
10. 선택 월 기준 CSV 다운로드
11. 카테고리별 소비 비중 시각화
12. 일자별 소비 추이 시각화
13. 예산 사용률, 예상 저축 가능 금액, 절약률 계산
14. 예산 80% 이상 사용 및 예산 초과 경고
15. Gemini 기반 스마트 소비 코치
16. 소비패턴 기반 카드 혜택 유형 추천 TOP 3
17. 전체 데이터 초기화

자동 결제 및 송금 기능
투자 상품 추천 및 자산 운용 기능

프리미엄 유료 결제 시스템

## KPI
- 소비 기록 유지율: 7일 이상 연속 소비 기록 사용자 수 / 전체 사용자 수 / 목표 70%
- 예산 설정률: 월 예산을 설정한 사용자 수 / 전체 사용자 수 / 목표 65%
- 예산 달성률: 예산 이하로 소비한 사용자 수 / 전체 사용자 수 / 목표 60%
- AI 피드백 조회율: AI 소비 분석 결과를 확인한 사용자 수 / 전체 사용자 수 / 목표 75%
- 과소비 감소율: 개선 전 대비 특정 과소비 카테고리 지출 감소 금액 / 기존 지출 금액 / 목표 20%
- 소비 분석 재방문율: 소비 분석 화면 재방문 사용자 수 / 전체 사용자 수 / 목표 55%
- 사용자 입력 완료율: 필수 소비 데이터 입력 완료 사용자 수 / 전체 사용자 수 / 목표 85%
- 절약 목표 달성률: 저축 목표를 달성한 사용자 수 / 전체 목표 설정 사용자 수 / 목표 50%
- OCR 인식 성공률: OCR 인식 성공 건수 / 전체 영수증 업로드 건수 / 목표 80%

# Experiment Plan

## 1. 실험 가설
AI가 소비 패턴 분석 결과와 절약 추천 문구를 함께 보여주면 사용자의 소비 분석 화면 재방문율과 예산 설정률이 증가할 것이다.

## 2. 실험군과 대조군
- Control: 소비 내역과 카테고리별 지출 데이터만 기본적으로 표시한다.
- Variant: 소비 분석 결과와 함께

“이번 달 배달비가 평균보다 30% 높아요”
“이번 주 커피 소비를 줄이면 5만원 추가 저축 가능”
같은 AI 맞춤 절약 피드백 문구를 함께 제공한다.

## 3. 주요 지표
AI 피드백 조회율
예산 설정률
소비 분석 화면 재방문율
절약 목표 설정률
카드 추천 조회율
카드 추천 클릭률

## 4. 성공 기준
Variant 그룹의 AI 피드백 조회율이 Control 대비 15% 이상 증가
예산 설정률이 10% 이상 증가
소비 분석 화면 재방문율이 10% 이상 증가
카드 추천 조회율 60% 이상
카드 추천 클릭률 30% 이상

## 5. 데이터 수집
expense_analysis_viewed 이벤트 수집
budget_created 이벤트 수집
ai_feedback_clicked 이벤트 수집
사용자별 feedback_exposed: boolean 속성 저장
소비 카테고리 및 예산 초과 여부 로그 저장
